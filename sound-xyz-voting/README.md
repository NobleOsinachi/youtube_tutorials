# Mina zkApp: Sound Xyz Voting

This template uses TypeScript.
See a full tutorial on youtube [here](https://youtu.be/6Jx7_7WForM)

## How to build

```sh
npm run build
```

## How to run tests

```sh
npm run test
npm run testw # watch mode
```

## How to run coverage

```sh
npm run coverage
```
